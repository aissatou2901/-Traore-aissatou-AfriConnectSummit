Classe :L1SEMI

 Description du projet

Site vitrine d'une conférence tech panafricaine fictive, « AfriConnect Summit »,
réalisé dans le cadre de l'examen de Technologie Web (Groupe ISI).
Le site présente l'événement, son programme sur 3 jours, ses intervenants
et permet l'inscription des participants.

Technologies utilisées

HTML5 (sémantique)
CSS3 (Flexbox, Grid, variables CSS, animations, responsive design)
JavaScript vanilla (aucun framework)
Git & GitHub / GitHub Pages

 Fonctionnalités JavaScript implémentées

 Dark mode / Light mode avec persistance localStorage
 Navbar dynamique au scroll + menu hamburger mobile
 Animations au scroll via IntersectionObserver
 Onglets du programme (Jour 1 / 2 / 3)
 Filtrage dynamique des intervenants
 Validation du formulaire d'inscription
 Bouton retour en haut
 Année dynamique dans le footer
 Compteurs animés au scroll
